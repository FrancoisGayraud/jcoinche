# jcoinche

REQUIREMENTS

    Maven
    JAVA JDK 1.8

COMPILE

    mvn package

CLEAN

    mvn clean

RUN

    Usage : java -jar target/jcoinche-server.jar
    Usage : java -jar [.jar] --ip [ip] --port [port] {--gui}

GUIDE

    Need four players/clients connected to play

    Follow instructions

    Colors: heart spade diamond club

    Values: ace, king, queen, jack, 10, 9, 8, 7
